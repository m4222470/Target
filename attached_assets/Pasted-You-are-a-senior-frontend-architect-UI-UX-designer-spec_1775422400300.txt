3. COLOR USAGE RULES (VERY IMPORTANT)

---

- Background must ALWAYS be dark (#0b0f1a)
- Neon colors are used ONLY for:
  • highlights
  • charts
  • active states
- Never fill large areas with neon colors
- Use gradients only for:
  • buttons
  • charts
  • key highlights

Gradient Example:
linear-gradient(135deg, #00f5ff, #7b61ff)

---

4. GLASSMORPHISM SYSTEM

---

Every card MUST follow:

background: rgba(255,255,255,0.05)
backdrop-filter: blur(10px)
border-radius: 16px
border: 1px solid rgba(255,255,255,0.1)

Shadow:
box-shadow: 0 8px 32px rgba(0,0,0,0.3)

---

5. GLOW SYSTEM (CRITICAL)

---

Apply glow ONLY to:

- active buttons
- chart lines
- important stats

Glow Example:
box-shadow:
0 0 10px #00f5ff,
0 0 20px rgba(0,245,255,0.5),
0 0 40px rgba(0,245,255,0.2)

Hover:

- Increase glow intensity
- Slight scale or translateY(-5px)

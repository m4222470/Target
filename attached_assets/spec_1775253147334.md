# Luxury Watch Product Page - Design Specification

## Project Overview

**Project Name:** Artisan Leather Watch Product Page
**Type:** E-commerce Product Landing Page
**Brand:** HOROLOGY CO.

## Design System

### Color Palette (Earth Tones - No Blue/Purple)
- **Primary:** Amber-600 (#d97706)
- **Primary Dark:** Amber-700 (#b45309)
- **Background Light:** Neutral-50 (#fafafa)
- **Background Dark:** Neutral-950 (#0a0a0a)
- **Text Light:** Neutral-900 (#171717)
- **Text Dark:** Neutral-100 (#f5f5f5)
- **Accent Green:** Green-600 (#16a34a)
- **Accent Gold:** Amber-500 (#f59e0b)

### Typography
- **Font Family:** System fonts with Lucide React icons
- **Headings:** Bold, tracking-widest for luxury feel
- **Body:** Regular weight, good line-height for readability

### Motion
- **Framer Motion** for all animations
- **Slide transitions** for gallery navigation
- **Fade-in-up** for content sections
- **Spring physics** for interactive elements

## Layout Structure

1. **Header:** Sticky navigation with logo, dark mode toggle, cart button
2. **Hero Banner:** Gradient background with brand messaging
3. **Main Content:** Two-column grid (Gallery | Product Details)
4. **Footer:** Brand information and links

## Components

| Component | Purpose |
|-----------|---------|
| Gallery | Product image carousel with thumbnails |
| ProductDetails | Name, description, features, specifications |
| ColorSelector | Color variant selection |
| QuantitySelector | Quantity input with +/- controls |
| AddToCartButton | Add to cart with animation feedback |
| DarkModeToggle | Theme switching |
| CartDrawer | Slide-out shopping cart panel |
